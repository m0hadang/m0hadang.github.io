
# parallelism
```text
CPU Core 1 -> Task 1
CPU Core 2 -> Task 2
```
독립적으로 여러 CPU 가 여러 작업들을 동시에 처리. 작업자 간 여러 작업들을 나누어서 처리.

# concurrency
```text
CPU Core
Task1 -> Task2 -> Task1 -> Task2
```

parallelism 와 달리 물리적으로 여러 동작을 동시에 처리하지 않고 큰 작업을 분할하여 동시에 처리.
